🐛 KNOWN ISSUES
════════════════════════════════════════════════════════════════════

1. Network Instability
   Impact: Slow dependency installation
   Workaround: Use yarn with increased timeout
   
2. Expo Module Versions
   Impact: Version selection prompts
   Workaround: Use SDK-compatible versions (55.x)
   
3. WebRTC Requires Native Build
   Impact: Cannot test in Expo Go
   Workaround: Must build dev client

4. Android mDNS Unreliable
   Impact: Auto-discovery may fail
   Workaround: QR code is primary method
